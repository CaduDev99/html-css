Atendimento Pré Hospitalar



É um serviço de emergência realizado fora do hospital. O APH é também conhecido como Atendimento Pré-Hospitalar.
Os objetivos do APH é Estabilizar a condição do paciente, Prevenir complicações, Transportar o paciente para uma unidade de saúde para receber o tratamento definitivo.
